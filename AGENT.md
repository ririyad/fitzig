## What is it?

A personalized fitness tracker

## What does it do?

- User can create a session
- A session contains:
    - Session Name (text)
    - No of Exercise (int)
    - Duration (hh:mm:ss)
- After creating session, user can select 
exercises from an existing list of exercises

- Based in no of exercise selected while creating a session, the user can select the number of exercises
- A user can define the progression of the exercise with repeat times
- When the user starts a session, a timer will appear and the session will continue based on the 'progression map' from the previous step
